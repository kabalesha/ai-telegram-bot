# 💈 Barbershop Booking Bot (Telegram MVP)

A simple Telegram bot that automates appointment booking for barbershops.  
Clients can choose a service, select a time slot, and enter their name.  
The barber receives instant notifications in Telegram.

---

## 🚀 Features

- ✂️ Service selection (Haircut / Beard Trim)
- ⏰ Time slot booking
- 👤 Client name collection
- 📩 Instant booking notifications to barber
- 💈 Simple and fast user flow
- ⚡ Lightweight MVP (no database required)

---

## 🧠 How It Works

1. User starts the bot in Telegram
2. Selects a service
3. Chooses a time slot
4. Enters their name
5. Barber receives a booking notification instantly

---

## 🛠 Tech Stack

- Node.js
- Telegraf (Telegram Bot Framework)
- Telegram Bot API
- JavaScript (ES Modules)

---

## 📦 Installation

```bash
npm install
```
